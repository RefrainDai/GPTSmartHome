# Custom Device Sounds

Place your own audio files here. The default mapping is in `config/media-map.json`.

Recommended files:

- `light_on.mp3`
- `light_off.mp3`
- `ac_on.mp3`
- `ac_off.mp3`
- `curtain_open.mp3`
- `curtain_close.mp3`
- `tv_on.mp3`
- `tv_off.mp3`
- `fan_on.mp3`
- `fan_off.mp3`
- `movie_mode.mp3`
- `sleep_mode.mp3`
- `welcome_home.mp3`

Supported by JavaFX in most environments: MP3 and WAV. If a file is missing, the app logs a warning and continues.
